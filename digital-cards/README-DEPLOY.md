# FluxGen Contact Cards - Cloudflare Pages

This repository contains the digital contact cards for FluxGen Industries Ltd.

## Files Structure
- `pratik.html` - Pratik Jhaveri's contact card
- `arpan.html` - Arpan Patel's contact card
- `abhishek.html` - Abhishek Patel's contact card
- `bhargav.html` - Bhargav Patel's contact card
- `static/images/optimized/` - Logo files

## Deployment
This is deployed to Cloudflare Pages as a static site.
